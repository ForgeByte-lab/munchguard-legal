# munchguard-legal

Static GitHub Pages site for MunchGuard's landing, support, privacy, and terms pages.

- `/` — landing page
- `/privacy-policy/` — Privacy Policy
- `/support/` — Support
- `/terms/` — Terms of Service

MunchGuard is a parent-controlled mealtime video app: kids watch parent-approved
videos only while they're actually chewing, detected on-device. No account, no
backend, no tracking.

Served via GitHub Pages from the `main` branch root (`.nojekyll` keeps paths verbatim).
